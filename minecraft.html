Welcome to Terra4

A casual Bedrock and Java compatible Minecraft server. Be kind, have fun, and read the rules before you play.

Server IP

mc.micjon.net

Version: 1.21.9

Server Rules

  1. No griefing. Respect other players' builds.
  2. No stealing or raiding without permission.
  3. No hateful or harassing language.
  4. No cheating or using exploits (x-ray, duping, hacked clients).
  5. Have fun and be respectful to staff and players.
  6. You will be banned with no more than 1 warning for cheating or griefing.

About

This server is meant for casual play. There is not heavy server managment so players who do not respect others' builds will quickly get banned.

Joining on Java

Open Minecraft -> Multiplayer -> Add Server -> Use IP: mc.micjon.net



Joining on Bedrock

Open Minecraft -> Multiplayer -> Add Server -> Use IP: mc.micjon.net and port 19009



Server is whitelisted for security, if you want to join for the first time join the discord and ping @mccakeah
